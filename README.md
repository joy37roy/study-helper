# AI Study Helper 📚

My second AI project — a study tool that explains any topic and quizzes you!

## Features
- Explains any topic simply using AI
- Generates quiz questions to test your understanding
- Saves every study session to a file
- Loops — study multiple topics in one session
- Type 'quit' to exit cleanly

## Technologies Used
- Python
- Groq API (llama-3.3-70b-versatile)

## What I Learned Building This
- Making multiple API calls in one program
- Using while True loops with clean exit
- Saving study sessions to files
- Building a complete multi-feature app

## How to Run
1. Install groq: pip install groq
2. Add your Groq API key
3. Run: python study_helper.py
